# spargo

Real-time GPU image dithering in the browser, via WebGL.

> *spargo* — Latin, "to scatter": dithering scatters pixels to simulate tone and shade.

**[Live demo →](https://gpu-dithering.vercel.app)**

## Features

- Real-time image dithering on the GPU, via WebGL
- Runs entirely in the browser, with no upload and no server
- Scatters pixels to simulate tone and shade
- Interactive live demo
